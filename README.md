## ✨ Features

* ⏱️ Pomodoro Timer (25 / 45 / 60 min)
* 🎨 Anime Companions (Arona, Plana)
* 🎵 Lo-fi Music Player
* 🏆 Achievements & Streaks
* 📊 Focus Tracking
* 🌙 Light / Dark Theme
* 📱 Fully Responsive
* 📲 PWA Ready (installable & offline)
